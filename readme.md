# E-Commerce Backend API

Production-style REST API built with Node.js, Express, MongoDB, and JWT Authentication.

## Tech Stack

- Node.js
- Express 5
- MongoDB
- Mongoose
- JWT
- bcrypt
- express-validator

## Features

- User Authentication
- Role-Based Authorization
- Validation
- Global Error Handling
- MVC + Service Layer Architecture
